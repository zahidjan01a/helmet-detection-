# Helmet Detection

A simple helmet detection project built with Python.

## Overview

This repository contains a small application that detects helmets in video input, such as from a webcam. The main file `webcam.py` is used to run the detection in real time.

## Files

- `Helmet_Detection.ipynb` - A Jupyter notebook containing project notes or experiments.
- `README.md` - Project description and usage information.
- `webcam.py` - Main Python script for running helmet detection with webcam input.

## Usage

1. Install required Python packages.
2. Run the webcam detection script:

```bash
python webcam.py
```

## Notes

- Make sure your environment has access to a working webcam.
- Add any required model weights or dependencies if needed.
